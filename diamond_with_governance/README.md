# Requirements

Run

```bash
yarn install
```

Install Ganache software [here](https://eips.ethereum.org/EIPS/eip-2535#upgrades-and-immutability)
and run it.

Specify network id in truffle-config.js. (You can find the network id in ganache).

# How to compile code

```bash
truffle compile
```

# Run test
```
truffle test
```
